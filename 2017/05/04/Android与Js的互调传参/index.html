准备工作

咱们这一篇讲的是Js+Android+原生webview的混合开发模式，

1.在需要展示js界面的原生界面创建一个WebView并为其设定初始设置

2.在android端的app/src/main/下创建assets资源文件目录，并将需要加载的html文件放在这个目录下边

3.在webview设置中添加加载本地文件代码

Js调用Android方法

Js调用Android方法，并向其传递参数（调用原生Toast方法，并向其传递显示的内容）

android端：
1.创建一个类用来专门存放需要被Js调用的方法集，其中的方法上一定要添加@JavascriptInterface注解，这个使用来向Js暴露原生中的方法的。

2.在设置webview的类中添加相应的设置

html端：
1.通过window的方式直接在标签事件属性中进行添加方法，其中test就是上一步在android端进行设置标签

Android与Js之间Json数据的传输与解析
